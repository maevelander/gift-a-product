Gift a product extension for Easy Digital Downloads

== Description ==

Simple extension for Easy Digital Downloads allowing users to 'gift a product'. Checkbox is added at checkout - when checked, the customer can enter gift recipient name, email and a gift message.

There are no plugin settings (for now), simply activate the plugin.

= Demo =

*	[Click here](http://demo.enigmaweb.com.au/edd-gift/) for out-of-the-box demo. The shop is in Test mode so no live transactions are processed.

= Languages =

Would you like to help translate the plugin into more languages? It's very easy - you don't have to know any code. Please [contact Maeve](http://www.enigmaplugins.com/contact/)


== Installation ==

1. Upload the `gift-a-product` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the WordPress 'Plugins' dashboard.
